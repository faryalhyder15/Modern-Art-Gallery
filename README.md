# Modern Art Gallery Website

A clean, responsive, and minimalist virtual art gallery built with **pure HTML & CSS**.  
Showcases contemporary artworks with a simple navigation structure, gallery view, and detailed artwork pages.

---

## Project Structure
modern-art-gallery/
├── index.html          # Home / Landing page
├── gallery.html        # Main gallery with artwork thumbnails
├── detail1.html        # Detail page - Blue Harmony
├── detail2.html        # Detail page - Texture Art
├── detail3.html        # Detail page - Dream Within a Dream
├── about.html          # About the gallery & featured artists
└── contact.html        # Simple contact form (mailto)

---


## Features

- Responsive top navigation bar (Home | Gallery | About | Contact)
- Centered, elegant layout using HTML tables for alignment (classic style)
- High-quality artwork thumbnails linking to detail pages
- Individual artwork detail pages with:
  - Large image
  - Title, year, medium, dimensions
  - Artist name
  - Short poetic description
- About page with mission statement and featured artists
- Contact page with basic form (uses `mailto:` – opens email client)
- Consistent styling: light background, centered content, horizontal rules

---

## Technologies Used

- **HTML5** – Structure & semantics
- **Inline CSS** – Simple styling (colors, alignment, fonts)
- **Base64 embedded images** – No external assets required (except gallery thumbnails)
- **mailto form** – Basic contact functionality without backend

---

## How to Run

1. Download or clone the repository
2. Open any `.html` file in a modern browser (Chrome, Firefox, Edge, Safari)
   - Recommended: start with `index.html`

No server needed — works completely offline.
